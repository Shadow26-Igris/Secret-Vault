Secret Vault is a secure credential management system designed to store, manage, and retrieve sensitive information such as API keys, tokens, and passwords in a centralized and encrypted manner.  
The project focuses on **security best practices**, **clean architecture**, and **real-world applicability**.

---

## Features

Secure storage of secrets using encryption
Authentication and authorization using JWT
Role-based access control
Centralized management of credentials
RESTful API for secure secret retrieval
Modular backend and frontend architecture



##  Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)

### Backend
- Node.js
- Express.js
- JWT Authentication
- Encryption utilities (for secure data handling)

### Database
- MySQL (or compatible relational database)
